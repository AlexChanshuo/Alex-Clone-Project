# Project Scope

## What Alex Clone Is

Alex Clone is a personal operating agent for Alex. It watches approved LINE
groups, remembers progress, summarizes social and project signals, keeps the
`alex-mind` Obsidian vault current, checks calendar context, and helps Alex
reply.

## Primary Jobs

1. **Group Intelligence**
   - Track important updates in each approved LINE group.
   - Identify questions directed at Alex.
   - Detect tasks, promises, deadlines, invitations, and relationship signals.
   - Report group-by-group progress daily.

2. **alex-mind Obsidian Vault**
   - Write daily group summaries.
   - Maintain people/project notes.
   - Save open loops and follow-up tasks.
   - Keep a daily diary note for Alex.

3. **Calendar Awareness**
   - Read Google Calendar after Alex connects it.
   - Warn about conflicts before suggesting replies or commitments.
   - Add proposed events only after confirmation.

4. **Reply Support**
   - Draft replies in Alex's style.
   - Ask before sending by default.
   - Auto-send only for explicit commands or approved low-risk patterns.

5. **Daily Briefing**
   - Morning: what needs attention today.
   - Night: what happened, what changed, what Alex should do tomorrow.

## Out of Scope for V1

- Reading private chats unless Alex explicitly approves them.
- Sending sensitive, financial, legal, medical, or relationship-heavy messages
  without confirmation.
- Pretending to be Alex in contexts where disclosure is required.
- Fully autonomous calendar creation.
- Deleting messages or modifying cloud permissions.
